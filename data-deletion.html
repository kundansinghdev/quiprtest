User Data Deletion – Quipr

Quipr allows users to request deletion of their personal data at any time.

How to request data deletion

  1. Send an email to manavhustles@gmail.com
  2. Use the subject line: “Quipr Data Deletion Request”
  3. Include the email or Instagram account used with Quipr

After verification, all associated data will be permanently deleted within 30 days.

Data deleted includes

  • User profile information
  • Instagram account connections
  • Automation rules
  • Leads and message logs

Contact: manavhustles@gmail.com
